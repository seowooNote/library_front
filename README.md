# 도서관리시스템 미니 프로젝트

## FRONT_END
### 폴더관리 개요
|폴더명|설명|
|--|--|
|apis|외부 API 모듈|
|atoms|전역상태관리 모듈|
|assets|이미지, PDF 등 참조 파일 모듈|
|components|컴포넌트 모듈|
|constants|상수 모듈|
|hooks|커스텀 훅 모듈|
|pages|페이지 모듈|

## BACK_END
### 폴더관리 개요
|폴더명|설명|
|--|--|
|aop|관점 지향 프로그래밍 모듈|
|config|Bean 수동 등록 모듈|
|controller|controller 모듈|
|dto|request, response dto(Client <-> Server)|
|entity|entity(Server <-> DB)|
|exception|예외 처리 모듈|
|repository|MySQL DB 모듈|
|service|service 모듈|
